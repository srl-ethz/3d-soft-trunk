Detailed Description

receives transform data for each frame from the OptiTrack system, which can then be used to calculate the current pose of the robot.

Acts as a client that communicates with the PC running Optitrack software. Uses the NatNetLinux library. Based on example_optitrack.cpp, which is based on SimpleExample.cpp provided as example in NatNetLinux.
