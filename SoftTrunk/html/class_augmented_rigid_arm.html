Detailed Description

Represents the augmented rigid arm model.

The rigid arm model approximates the soft arm. (see paper etc. for more info on how this is so) This class can calculate the kinematics & dynamics of the augmented arm using RBDL. It can also generate XACRO files based on the parameters of the robot.
