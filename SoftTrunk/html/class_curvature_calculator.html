Detailed Description

Calculates the current configuration(deltaLx, deltaLy of each segment) of each soft arm segment based on the OptiTrack measurements of the base and tip of a segment.

The frames have to be set up properly in Motive first. id conventions: base of robot is 0, frame that is between first and second segment is 1, and so on....
